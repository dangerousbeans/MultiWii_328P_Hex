This Archive contains a ready-patched Version of
MultiWii 2.1 with all neccessary modifications to
utilize the Hobbyking 
>> MultiWii 328P Flight Controller w/FTDI & DSM2 Port <<
on a X-Quadcoter.

Just load into the ARDUINO SDK and upload to the board.

Please note, that the PIN-Names printed on the board
are partly wrong.
See the attached pictures in the picture folder for 
the correct PIN-Names following the MultiWii-Schema.

For more information please visit:

www.flugwiese.de
